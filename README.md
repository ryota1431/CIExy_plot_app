# CIExy色度図プロットアプリ

CIExy色度図上に任意の点とラベルをプロットできるWebアプリです。SVG形式でダウンロードも可能です。

---

## 主な機能

- CIExy色度図の表示
- x, y座標・色・ラベル指定によるプロット
- SVG画像のダウンロード
- レスポンシブデザイン対応

---

## 使い方

1. フォームに座標・色・ラベルを入力
2. 「プロット」ボタンで色度図に点とラベルを追加
3. 「SVGダウンロード」ボタンで画像を保存
